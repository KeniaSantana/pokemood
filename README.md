## tituilo de la aplicacion
* "Pokemood" *

  ## Objetivo de Pokemood
  Esta aplicacion tiene de proposito de desarrollar una aplicación multiplataforma llamada Pokemood utilizando Flet y Python que permita a los usuarios registrar sus estados de ánimo mediante la asignación de Pokémon relacionados con sus emociones.
